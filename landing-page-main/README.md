# Mechanism of Action Prediction Dashboard

Live website: [https://landing-page-main-two.vercel.app](https://landing-page-main-two.vercel.app)

This is a Next.js dashboard for exploring mechanism-of-action prediction workflows. It includes upload, preview, dataset insight, charting, and CSV download flows through Next API routes.

## Getting Started

Install dependencies and run the development server:

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Build

```bash
npm run build
npm run start
```
